            # Nâng cấp Auto Accept: Phân loại Context & Cấu hình Chi tiết
            Cải tiến chức năng Auto Accept để hỗ trợ phân loại dialog (Terminal, Review, System), cho phép bật/tắt từng loại và gán bộ lọc context riêng cho từng nhóm, tương tự như độ nghiêm ngặt của Auto Retry.
            ## User Review Required
            > [!IMPORTANT]
            > Cấu trúc file `config.json` sẽ thay đổi từ dạng phẳng sang lồng nhau cho `autoAccept`. Tuy nhiên, code sẽ hỗ trợ fallback để không làm hỏng các bản cài đặt cũ.
            ## Proposed Changes
            ### [Core] config.json
            #### [MODIFY] [config.json](file:///Users/lehoangthang/Documents/antigravity-auto-click/config.json)
            - Chuyển đổi `autoAccept: true` thành object chứa `enabled` và `categories`.
            - Thêm các nhóm mặc định: `terminal`, `review`, `system`.
            ### [Payload] src/injection-payload.js
            #### [MODIFY] [injection-payload.js](file:///Users/lehoangthang/Documents/antigravity-auto-click/src/injection-payload.js)
            - Cập nhật logic `scanAndAction`:
              - Thay vì chỉ check `actionButtonPatterns`, sẽ duyệt qua danh sách các category được bật.
              - Mỗi category sẽ có bộ patterns riêng.
              - Luôn kiểm tra context dialog so với patterns của category trước khi click.
              - Riêng nhóm `terminal` sẽ giữ lại logic check `blacklist`.
            ### [CLI/UI] src/extension.js & scripts/menu.sh
            #### [MODIFY] [extension.js](file:///Users/lehoangthang/Documents/antigravity-auto-click/src/extension.js)
            - Cập nhật UI menu để hiển thị danh sách các sub-category của Auto Accept.
            ## Verification Plan
            ### Automated Tests
            - Chạy `node scripts/trigger-accept-test.js` để kiểm tra khả năng nhận diện với cấu trúc mới.
            ### Manual Verification
            - Kiểm tra việc bật/tắt từng loại trong config xem hệ thống có phản ứng đúng không.
